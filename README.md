# Bugfixing: Těžší úloha

Úkolem je nalézt a opravit chyby v aplikaci pro konverzi CSV souborů do Excelu (XLSX).

## Úkoly:

1. Naklonovat tento repozitář s Maven projektem.

2. Opravit chyby v aplikaci tak, aby generovala stejný výstup jako expected_result.xlsx
   - podstatná je datová konzistence a pořadí
   - velikost a sloupců a buněk není předmětem úkolu

3. Do nové větve nahrát opravu a založit merge / pull request.
